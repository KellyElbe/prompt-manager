# 热点运营常用prompt

这是一个专为内容创作团队设计的 AI Prompt 管理系统。

## 📁 项目结构

### 🔧 后台管理
- **文件位置**: `后台管理/ai-prompt-manager.html`
- **功能**: 管理员专用，可以添加、编辑、删除 prompt 和分类
- **权限**: 完整的编辑权限

### 👥 前台展示  
- **文件位置**: `前台展示/prompt-frontend.html`
- **功能**: 团队成员使用，只能查看和复制 prompt
- **权限**: 只读权限

## 🚀 使用方法

1. **管理员使用**：打开 `后台管理/ai-prompt-manager.html` 进行内容管理
2. **团队成员使用**：打开 `前台展示/prompt-frontend.html` 查看和复制 prompt
3. **数据同步**：两个页面的数据会自动同步，管理后台的修改会实时反映到前台

## ✨ 主要功能

- 📝 分类管理（支持自定义分类）
- 🔍 内容搜索和筛选
- 📋 一键复制功能
- 🔄 实时数据同步
- 📱 响应式设计
